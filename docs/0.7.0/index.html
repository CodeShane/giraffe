Giraffe¶

This is the documentation for version 0.7.0 of Giraffe. For older or newer versions, see the main site.

Installing¶

Giraffe is published as a Maven-style artifact, but we recommend using Gradle for builds:

repositories {
    mavenCentral()
    maven {
        url 'http://dl.bintray.com/palantir/releases'
    }
}

dependencies {
    compile 'com.palantir.giraffe:giraffe-core:0.7.0'
    compile 'com.palantir.giraffe:giraffe-ssh:0.7.0'
}
