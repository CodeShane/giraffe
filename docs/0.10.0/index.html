Giraffe¶

This is the documentation for version 0.10.0 of Giraffe. For older or newer versions, see the main site.

Installing¶

Giraffe is published as a Maven-style artifact, but we recommend using Gradle for builds:

repositories {
    jcenter()
}

dependencies {
    compile 'com.palantir.giraffe:giraffe-ssh:0.10.0'
    // or 'com.palantir.giraffe:giraffe-core:0.10.0' for local features only
}
